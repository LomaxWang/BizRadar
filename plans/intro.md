## 💡 为什么你需要 IdeaHunter？

在大模型时代，写代码不再是壁垒，\*\*“缺乏商业嗅觉，不知道做什么能赚钱”\*\*才是独立开发者最大的痛点。

你是否经常陷入这样的困境：

  * ❌ 满脑子前沿技术（RAG、Next.js），却找不到落地的场景？
  * ❌ 辛辛苦苦熬夜一个月做出的产品，上线后零访问、零付费？
  * ❌ 混迹于技术社区，根本不知道小红书博主、闲鱼卖家、房产中介每天在为什么痛苦？

**IdeaHunter 就是为你解决“信息茧房”而生的。** 它不是一个简单的爬虫，而是一个拥有“毒舌投资人”思维的多智能体系统。它替你潜入全网各个下沉市场和非技术圈层，每天为你过滤出真正有付费意愿、且大厂绝对不会去做的微型商业机会。

## 🚀 核心功能 (Core Features)

  * 📡 **全域商机雷达 (Omni-Channel Radar)：**
    内置多平台插件（默认支持 V2EX、知乎、小红书特定话题），24小时不间断巡航，专门捕捉带有“太痛苦了”、“求推荐工具”、“每天都要手动”等高频抱怨的文本。
  * 🧠 **AI 毒舌评审委员会 (AI Critic Board)：**
    抓取到抱怨后，不会盲目推送。内置的“产品经理 Agent”和“投资人 Agent”会进行内部辩论，严格套用\*\*“高频发生 + 大厂不做 + ROI清晰”\*\*的三大黄金法则进行漏斗过滤，剔除 95% 的伪需求。
  * 🖨️ **一键生成立项书 (Turnkey PRD Output)：**
    对于通过层层考验的优质点子，直接输出保姆级的《微型产品立项书》，包含：需求溯源、付费意愿评估、竞品分析、技术栈推荐及冷启动获客建议。

## 🛠️ 快速开始 (Quick Start)

IdeaHunter 主打开箱即用，只需 3 行命令即可唤醒你的赛博商业合伙人：

**1. 安装依赖**

```bash
git clone https://github.com/yourusername/IdeaHunter.git
cd IdeaHunter
pip install -r requirements.txt
```

**2. 配置环境变量**
复制 `.env.example` 文件并重命名为 `.env`，填入你的大模型 API 密钥（推荐使用 DeepSeek 或 Kimi，成本极低）：

```env
LLM_API_KEY="sk-your-api-key-here"
LLM_BASE_URL="https://api.deepseek.com/v1"
```

**3. 启动雷达**

```bash
python main.py --target v2ex --mode daily_report
```

*喝杯咖啡，等待终端为你打印出今天的财富密码吧！*
